          Manage environments

          Environments help you customize requests according to variables. Variables can be used in the following form - {{variable}}. The string {{variable}} will be replaced with its corresponding value. For example, for an environment variable 'url' with the value 'http://localhost' , you will have to use {{url}} in the request URL field. {{url}} will be replaced by http://localhost when the request is sent.
